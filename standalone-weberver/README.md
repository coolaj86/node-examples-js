You Got ServeD!
===

Standalane one-liner HTTP webserver for NodeJS

Equivalent of `python -m SimpleHTTPServer 3000`

Installation
===

    npm install -g served

Usage
===

    cd /path/to/serve
    served [port]  # defaults to 3000

example:

    cd ~/Downloads
    served 5555
